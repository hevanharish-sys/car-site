# Xtrme

A modern, high-performance web application for automotive discovery and interaction, built with React, TypeScript, and Vite.

## 🚀 Features

- **Blazing Fast**: Leverages Vite for instant server start and lightning-fast HMR.
- **Type Safe**: Fully written in TypeScript for robust development and maintainability.
- **Modern UI**: Styled with Tailwind CSS for a sleek, responsive, and customizable design.
- **Iconography**: Uses Lucide React for consistent and beautiful icons.

## 🛠️ Tech Stack

- **Framework**: [React 19](https://react.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Icons**: [Lucide React](https://lucide.dev/)

## 📦 Getting Started

### Prerequisites

- Node.js (Latest LTS version recommended)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/hevanharish-sys/car-site.git
   ```
2. Navigate to the project directory:
   ```bash
   cd xtrme
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Development

Start the development server:
```bash
npm run dev
```

### Build

Create a production-ready build:
```bash
npm run build
```
